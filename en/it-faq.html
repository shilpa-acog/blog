How can I ssh into a remote server?

If you are logging in using ssh into a server for the first time, you need to mention the port number of the server and add the username too. The command structure is:
ssh -p <port> username@servername.aganitha.ai
For example:
ssh -p 2222 abhijeet@own2.aganitha.ai After Connecting once like this, next time you dont need to mention the port number and remaining. just you need to connect by
ssh own2
